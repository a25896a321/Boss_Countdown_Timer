該程式是一款專為遊戲輔助設計的倒數計時工具，支援多組獨立計時、自定義音效及透明介面，幫助玩家精準掌控 BOSS 機制與技能循環。

This program is a precise countdown tool designed for gaming assistance. It supports multiple independent timers, custom sound effects, and a transparent interface, helping players master BOSS mechanics and skill cycles.


🛡️ 重要

設定系統管理員權限

方法 1：右鍵程式執行檔 → 選擇「系統管理員執行」。

方法 2：右鍵程式執行檔 → 選擇內容 → 選擇相容性 → 勾選「系統管理員執行」。

簡易使用說明

→開啟計時器模式	（F1）

→火眼召喚出特效	（Num /）

→本體切換模式	（Tab）

→遇到機制魔方重置	（`）

→遇到延遲+時	（CapsLock） 

→打完後關閉	（Num *）


結論：

1. 該出黑水、魔方卻沒出！ → 小招加時5s、召喚加時10s

2. 火眼召喚按 " / "　　進本體按 " TAB "　　遇到機制魔方按 " ` "

3. 本體 3 階建議減少加時動作

4. 本體 3 階的機制魔方是單顆


🖥️ 主介面說明

置頂及語言：設定置頂避免被擋住、調整中英語言。

起始計時器：亮綠色顯示，適合固定間隔事件，具備自動循環功能。

回合計時器：雙階段切換。回合 1（粉紫色）結束後，自動切換至回合 2（亮綠色）。

透明度：透過拉桿可調整 5% - 100% 透明度。


⌨️ 操作快捷鍵（熱鍵）

Num /	啟動起始計時：啟動後會自動暫停回合計時器

`	開始/重置回合：每次按下皆會從回合 1 重新開始

Num *	全部停止：立即停止所有計時

CapsLock	補時：針對「運行中」的計時器微調+時間

Num -	扣時：針對「運行中」的計時器微調-時間

F3	備註筆記：置頂筆記區，紀錄你需要的文字

F2	站位方塊標記：顯示１塊畫布，用於標記玩家站位

F1	計時器模式：切換至精簡介面，不擋遊戲

Num .	buff提示：每次按下技能（請設置與熱鍵對應），後台會倒數時間到，並播放音效提示

Num +	時間筆記：此功能無音效提示，可設定時間鐘，並操控管理該時間
